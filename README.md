# Analyse — Méthode Entité-Relation

Ce dépôt contient des notes et exercices liés aux systèmes de gestion de bases de données (SGBD), inspirés du document fourni `SGBD.pdf` présent à la racine du projet.

**Contenu principal**
- Introduction aux SGBD et concepts fondamentaux
- Modèle Entité-Relation : entités, attributs, relations, cardinalités
- Passage du modèle ER au modèle relationnel
- Normalisation (1NF, 2NF, 3NF) et exemples
- Schémas relationnels et intégrité référentielle
- Commandes SQL d'exemples (création, requêtes, jointures)
- Exercices corrigés et cas pratiques
